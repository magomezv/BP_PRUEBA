## ⚙️ Preparación de ambientes SQL y PySpark
- Dar clic en el botón **play** ubicado a la izquierda para configurar el ambiente de compilación.  
- Verificar que el código se haya ejecutado en su totalidad.  
- Revisar que aparezca el mensaje:  
  **"CONFIGURACION DE AMBIENTE SQL y SPARK FINALIZADO"**  
- Ejecutar la sección que reinicia el ambiente (aparecerá un mensaje de fallo de sesión, pero es normal porque el reinicio es necesario).  
- Validar la versión.  
- Una vez finalizada la configuración, se pueden validar las respuestas en las dos secciones.
